$가속도(시간에 따라 속도가 변하는 량) -> 어느정도 기울어 졌는지 '각도' 측정
('중력가속도'를 이용하여 어느 각이 어느정도 받느냐를 이용하여)
즉, z축과 다른 축의 값을 atan2(아크탄젠트로 추정)연산하여 '연직방향(중력가속도가 작용하는 방향)'에 비하여
얼마나 기울어 졌는지 측정

#센서를 default 상태로 두었을 경우, x,y = 0이고 z값 = 16383
순간순간의 값에서 치명적 -> " 매 순간순간 데이터는 확신할 수 없지만 평균적인 값은 신뢰할 수 있음 "

$자이로(각속도)  : 특정 축을 기준으로 회전하는 속도를 나타냄(적분 시, 각도를 얻을 수 있음)
* 시간의 경과로 드리프트(적분 오차)의 발생으로 각도가 틀어질 수 있음

#초기값과 비교하여 현재 각도를 알 수 있음(미소 시간간격 dt마다 돌아가는 while문안에서 angle에 각속도값을 더해줌 -> 적분)
-> " 매 순간순간 값은 보다 정확하지만 평균적인 값은 신뢰할 수 없음"

가속도 센서와 자이로 센서의 장점을 합쳐 이상적인 값을 얻음 -> complementary filter

상보필터 공식 -> angle = 0.98 * (angle + gyroData * dt) + 0.02 * accData
 // 0.98과 0.02는 사용자가 임의로 조정가능한 파라미터이며, 가속도쪽에 값이 작게 곱해진것은 가속도계의 순간적인 영향력은 줄이되, 미소하게 자이로 값에 개입할 수 있도록 최적화한 것
angle(각도 값)을 매번 업데이트
